GO функциональный анализ завершен.
Метод: over-representation analysis (гипергеометрический тест) по GO-терминам из GAF.
Коррекция множественного тестирования: BH (qvalue).
Порог значимости термов: qvalue < 0.05.
Визуализация: горизонтальные столбчатые диаграммы top-термов по -log10(qvalue) отдельно по онтологиям.
Дополнительно выполнен GO-анализ для пересечений и уникальных наборов DEG из Venn_results; результаты сохранены в Enrichment_results/GO/CommonUnique_GO.
